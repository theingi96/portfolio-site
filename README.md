# Portfolio Site

## Overview

A personal portfolio website built with Ruby on Rails to introduce myself, showcase my projects, skills, and work experience.

This portfolio was created for my job search as a Web Engineer in Japan.

---

## Features

* Home page
* About Me
* Skills
* Projects
* Experience
* Contact
* Resume Download
* Responsive Design
